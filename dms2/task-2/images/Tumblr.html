      what the task asked were to create a simple html page with a heading, two paragraph and three buttons that do some creative stuff with javascript, but that was a little bit boring so I make it look a little bit funnier. i made it look like a text editor with the exact same code for the task.

      the three buttons is the code for it itself, I had fun making them.

      preview | download

      External image

      i have experience with the jquery library but not much because my specialty was in html, css and php so although i have knowledge to create a decent site, it is still kind of a challenge for me.

      i used to just using other people script with a little modify to fit my need but i never try to create a new one, so i hope this course gonna help me improve that a bit.

      my first try with dom manipulating is to replicate the nyan.cat site, i wanted to make a running cat and a button to stop the cat from running and vice versa but quickly realize what i did have nothing to do with the task so i change my design into the current one.

      coding the front-end look of the site was kinda easy for me, but scripting the button is not. after a while, i finally decided to make three buttons.

      one change the style of the editor, one change the background color and the last one just make an alert box because i dont know what to do with it.

      the first button i use dom to detect the source link of the css file and change it to another one with a click, but i also use an if to check if when the button is click, the link is the original one or not to make it a toggle button, change the file back and forth instead of just once.

      the second button i also use dom to change the color of the background css. at first, i tried to switch between some previously decided color, but luckily found a script made by paul irish to create random hex color value.

      the third one is just a simple alert box.
